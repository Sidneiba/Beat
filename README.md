# Mozart

Bem-vindo ao projeto "Mozart", uma iniciativa ambiciosa para explorar os limites da geração de música por computador. Inspirado pelo gênio de Wolfgang Amadeus Mozart, este projeto busca criar um sistema colaborativo capaz de compor peças musicais originais e cativantes.

## Sobre o Projeto

"Mozart" é um sistema modular que utiliza inteligência artificial e algoritmos de composição para gerar música de forma autônoma. O projeto é dividido em diversos componentes, cada um com uma função específica:

* **Gerador Aleatório de Partituras:** Cria sequências musicais iniciais de forma aleatória.
* **Identificador de Trechos Musicais:** Analisa as partituras em busca de padrões e trechos interessantes.
* **Combinador Harmônico:** Combina os trechos identificados, criando progressões harmônicas.
* **Avaliador e Combinador:** Avalia as composições geradas e combina as melhores para criar novas variações.
* **Feedback do Usuário:** Permite que os usuários avaliem e forneçam feedback sobre as músicas geradas.

## Começando

Para começar a explorar o projeto "Mozart", siga estas etapas:

1.  Clone este repositório para o seu computador:

    ```bash
    git clone URL_DO_SEU_REPOSITORIO
    ```

2.  Navegue até o diretório do projeto:

    ```bash
    cd Mozart
    ```

3.  Instale as dependências necessárias:

    ```bash
    pip install -r requirements.txt
    ```

4.  Execute o gerador de partituras:

    ```bash
    python gerador_partituras.py
    ```
